# warmup-first-repo

Warm-up repository created as part of Engineering Onboarding. It documents my progress learning Git & version control, understanding the FloCard product and release process, and setting up my development environment.

## 📌 Purpose

This repository is the hands-on deliverable for the onboarding task — proof of working Git usage (staging, committing, pushing) alongside written notes from each day of onboarding.

## 📁 Contents

| File | Description |
|---|---|
| `README.md` | This file — repository overview and index. |
| `day1-summary.md` | Day 1 learning summary (team, environment, first impressions). |
| `day2-product-understanding.md` | Day 2 product understanding summary (FloCard features, users, use cases). |
| `day3-git-notes.md` | Day 3 Git learning notes (concepts, commands, branching strategy). |

## 🌿 Branching Strategy (FloCard)

- **main** — production-ready code only
- **dev** — integration branch for completed, tested features
- **feature/\*** — individual feature/fix branches, created off `dev`
